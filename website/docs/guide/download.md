# 如何下载

RTSBuilding 可以从 Modrinth 或 CurseForge 下载。

## 官方下载页

- [Modrinth：RTS building: Build From Above](https://modrinth.com/mod/rts-building-build-from-above/versions)
- [CurseForge：RTS building: Build From Above](https://www.curseforge.com/minecraft/mc-mods/rts-building-build-from-above)

进入下载页后，选择与你的 Minecraft 版本和加载器匹配的文件。

## 支持版本

目前主要支持：

- Minecraft `1.21.1` / NeoForge
- Minecraft `1.20.1` / Forge

如果你在服务器或整合包中使用，请以服务器或整合包要求的 Minecraft 版本和加载器为准。

## 在启动器里搜索

如果你使用 PCL 或 HMCL，也可以在启动器的模组下载页面搜索：

```text
RTSbuilding
```

如果搜不到，试试：

```text
RTS building
```

或：

```text
RTS building: Build From Above
```

下载时同样要确认 Minecraft 版本和加载器是否匹配。

## 手动安装

如果你从网页下载 `.jar` 文件，把它放进对应实例或整合包的 `mods` 文件夹。

启动游戏后，按 `G` 打开 RTSBuilding。
