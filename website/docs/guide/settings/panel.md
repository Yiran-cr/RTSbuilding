# 设置面板

点击 RTS 界面右上角的齿轮按钮，打开设置面板。


## RTS UI 缩放

点击面板上的 `-` / `+` 按钮，调整底部栏、顶部栏和悬浮窗的界面大小。

调整此选项只会改变 RTSBuilding 面板的大小，不会改变原版物品栏等界面的缩放比例。


## 灵敏度

点击面板上的 `-` / `+` 按钮，调整 RTS 镜头移动、旋转和缩放的速度。

调整后在 RTS 视角内拖拽测试效果。

## 自动存入

打开“自动存入”后，RTS 模式里挖下来的物品会优先进入绑定储存。
关闭后，挖掘产物直接掉落原地。

整理背包或防止物品存错时，请关闭此项。

## Shift 存入

打开“Shift 存入”后，在背包、箱子或机器界面中，按住 `Shift` 再右键点击物品，物品会直接进入当前绑定的 RTS 储存。

适用于大批量搬运：刚连接 AE 系统后，打开 Shift 存入，把箱子里的材料快速送进 AE；整理基地旧仓库、清空施工材料箱、集中机器产物时也很适合。

关闭后，`Shift + 右键` 不再触发 RTS 存入。

## 容器悬浮窗

打开“容器悬浮窗”后，打开箱子、工作台或机器时，界面旁边会显示 RTS 侧边悬浮窗。
关闭后隐藏侧边悬浮窗。


## 镜头设置

常见选项包括：

- 镜头从玩家头顶开始。
- 平滑 RTS 镜头。
- 水平拖屏反向。
- 垂直拖屏反向。

若拖拽方向不适，切换对应的反向选项。
若镜头移动生硬，开启“平滑 RTS 镜头”。

## 伤害提示

生存模式下开启“受击音效”或“低血量自动退出 RTS”，避免玩家本体在 RTS 视角下遇袭却没有察觉。

## 服务器规则

建造范围、材料消耗、技能解锁等多人生存规则以服务器或模组配置为准。

更详细的生存解锁和家园范围说明，请看“生存模式机制”。

整合包作者和服务器管理员可以继续查看“整合包与服务器配置”。
