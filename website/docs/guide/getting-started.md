# 开始使用

## 打开 RTSBuilding

游戏内按 `G`（默认键）打开 RTSBuilding，进入俯视建造视角。

再次按 `G` 或 `Esc` 关闭 RTS 界面。


## 界面布局

RTSBuilding 的界面主要分为三块：

- 顶部栏：切换模式，打开快速建造、连锁挖掘、设置和教程。
- 底部栏：查看储存物品、选择方块、使用蓝图、远程合成。
- 侧边悬浮窗：打开箱子或机器界面时出现，用于将容器物品存入 RTS 储存。

## 首次使用指南

第一步：绑定一个箱子作为 RTS 储存。

1. 打开 RTSBuilding。
2. 点击顶部栏的储存绑定按钮。
3. 准星对准场景中的箱子。
4. 左键点击箱子。
5. 确认顶部提示显示“已绑定储存”。
6. 确认底部栏已显示箱内物品。


绑定后，左键点击底部栏的方块，准星对准放置位置并右键点击。

## 生存模式限制

若界面缺失按钮，说明已开启生存平衡。打开背包，点击屏幕顶部的“RTS 技能树”按钮，用材料逐步解锁储存绑定、远程放置、远程挖掘等能力。

生存平衡下还需要设置 RTS 家园。打开背包，点击屏幕顶部的“RTS 家园”按钮，设置家园位置；RTSBuilding 会以家园为中心决定可使用 RTS 功能的范围。

详细机制请看“生存模式机制”里的“技能树与能力解锁”和“RTS 家园与范围限制”。

## 教程与提示

点击界面各区域的 `i` 按钮即可查看对应教程。

顶部提示会实时显示当前模式和下一步操作要求，例如已选物品、绑定储存状态、快速建造等待 A/B 点。
