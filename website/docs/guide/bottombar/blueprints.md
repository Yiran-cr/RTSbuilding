# 蓝图空间

蓝图空间用于导入、选择、预览和放置建筑蓝图。

你可以使用已有蓝图，也可以在世界中框选建筑，保存成新的蓝图。


## 打开蓝图空间

打开 RTSBuilding 后，看底部栏上方的标签页。

点击“蓝图空间”，底部栏会切换到蓝图界面。

如果服务器或整合包关闭了蓝图功能，界面会提示蓝图不可用。

## 导入蓝图

点击“上传”按钮，选择蓝图文件。

支持的格式包括：

- `.nbt`
- `.schem`
- `.schematic`
- `.litematic`

导入成功后，蓝图会出现在列表里。点击蓝图名称即可选择。

如果导入失败，先确认文件格式是否支持，文件是否过大，以及蓝图里是否包含当前游戏没有的方块。


## 同步机械动力蓝图

如果你使用机械动力的蓝图，可以点击“同步机械动力”。

RTSBuilding 会从当前游戏文件夹中读取已有的机械动力蓝图，并复制到 RTSBuilding 的蓝图文件夹。

同步完成后，刷新蓝图列表即可选择这些蓝图。

## 选择和预览蓝图

点击蓝图列表中的一个蓝图，世界中会出现预览。

预览用于确认位置、朝向和大致形状。先移动镜头，找到合适的放置位置。

如果没有看到预览，先确认你已经选中了蓝图，并且鼠标指向了可以放置的世界位置。

选中蓝图后，按 `X` 可以清除当前蓝图预览。这个按键可以在 Minecraft 的按键设置中修改。


## 固定预览

对准世界中的方块右键，把蓝图预览固定在当前位置。

固定后，蓝图不会跟着鼠标移动。先检查位置，再决定是否放置。

如果预览位置不对，清除预览后重新固定，或使用移动按钮微调。


## 移动、升降和旋转

蓝图固定后，使用蓝图空间里的移动和旋转按钮调整预览。

常见操作包括：

- 点击屏幕里的方向按钮，向前、向后、向左、向右移动蓝图。
- 使用上升/下降按钮调整高度。
- 使用旋转按钮调整蓝图朝向。
- 键盘方向键用于水平微调。
- `Page Up` / `Page Down` 用于上下微调。

调整后，看世界中的预览是否对齐。确认无误后再放置。

## 放置蓝图

确认位置、朝向和材料后，点击“放置”按钮开始建造。

生存模式下，RTSBuilding 会检查材料是否足够。缺少材料时，蓝图空间会显示缺少哪些材料。

创造模式下不需要消耗材料。


## 材料检查

蓝图空间会显示蓝图需要的材料，以及当前能建造多少。

如果缺材料，查看缺少的方块名称和数量。补齐材料后，再回到蓝图空间刷新或重新选择蓝图。

如果蓝图里有当前游戏没有的方块，界面会把它们标出来。能放置的部分仍然可以继续使用。


## 保存建筑为蓝图

点击“创造新蓝图”，进入框选模式。

基本步骤：

1. 在世界中右键选择第一个角。
2. 再右键选择对角。
3. 检查框选范围。
4. 如果有不想保存的方块，按提示排除。
5. 点击保存，并给蓝图命名。

框选过程中按 `X` 可以取消本次框选。

保存后，新蓝图会出现在蓝图列表中。


## 重命名、删除和另存为

选中蓝图后，可以重命名或删除。

如果要把蓝图分享给别人，使用另存为，把蓝图保存到你选择的位置。

删除前请确认蓝图不再需要。删除后，它会从蓝图文件夹中移除。
