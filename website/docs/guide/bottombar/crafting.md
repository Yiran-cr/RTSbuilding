# 合成面板

合成面板用于在 RTSBuilding 中查看和制作物品。

它会根据当前可用材料显示可合成的物品。你可以不用离开 RTS 视角，就从绑定储存里制作常用方块或物品。


## 打开合成面板

在屏幕下方的底部栏左侧，可以找到合成按钮。

解锁合成终端后，点击带有 `C` 字样的按钮，或者直接按键盘上的 `C` 键，可以打开合成面板。

如果没有看到合成入口，通常表示这个功能还没有解锁。

## 搜索可合成物品

打开合成面板后，在搜索框里输入物品名称。

列表会显示符合搜索的可合成物品。点击物品后，可以选择制作数量。

如果列表里没有你想要的东西，可以先清空搜索，再确认当前绑定储存里是否有足够材料。

## 选择合成数量

右键点击列表里的物品图标，会弹出数量选择窗口。

你可以输入想制作的数量，或使用界面里的按钮调整。

确认后，RTSBuilding 会从当前可用材料中制作物品。做好的物品会自动放进你的背包，或者存入绑定的箱子里。


## 缺少材料

如果材料不够，合成面板会提示缺少哪些材料。

这时可以先回到底部栏的储存空间，检查材料是否在绑定储存里。补齐材料后，再回到合成面板制作。


## JEI 联动

如果安装了 JEI，你可以先在 JEI 里查看目标物品的配方，再回到 RTSBuilding 的合成面板搜索这个物品。

只要是支持的配方，合成面板上就会显示它能否制作、还缺什么材料。

如果搜不到想要的物品，请检查箱子里有没有放对材料、这个物品在当前游戏里能不能合成，以及输入的物品名字有没有打错。
