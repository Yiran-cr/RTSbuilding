# 底部栏总览

底部栏位于 RTSBuilding 界面的最下方。它是选择材料、查看储存、使用蓝图和远程合成的主要区域。

底部栏内容比较多，阅读时可以先把它分成几块：左侧控制区、储存空间、创造物品栏、蓝图空间和合成面板。


## 底部栏分成哪些部分

- 左侧控制区：调整排序方式、底部栏高度，打开合成入口，绑定远程机器界面。
- 储存空间：查看可用物品、玩家快捷栏、RTS 快捷槽、最近使用和流体储存。
- 创造物品栏：创造模式专用，从原版和模组创造标签页中直接选择 RTS 放置物品。
- 蓝图空间：导入、选择、预览、移动、旋转、放置和保存蓝图。
- 合成面板：搜索可合成物品，并直接消耗绑定储存里的材料制作。

底部栏上方还有标签页。点击“储存空间”“创造”或“蓝图空间”，可以在物品储存、创造选材和蓝图功能之间切换。“创造”标签只会在创造模式下显示。


## 先看哪一篇

如果你只想知道底部栏每个按钮在哪里，先看“左侧控制区”。

如果你想使用方块、工具、快捷槽、最近使用或流体，查看“储存空间”。

如果你在创造模式下想直接从创造标签页选材，查看“创造物品栏”。

如果你要导入建筑蓝图、固定预览、调整位置并放置，查看“蓝图空间”。

如果你要从绑定储存里直接制作物品，查看“合成面板”。

## 悬停提示

把鼠标放在物品、流体或按钮上，会显示对应提示。

物品提示会出现在底部栏左侧附近，方便查看附魔、耐久、模组物品说明等原版提示。

如果你不确定某个格子是什么，先悬停查看提示，再点击操作。
