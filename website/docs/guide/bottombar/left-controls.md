# 左侧控制区

左侧控制区位于底部栏最左边，主要负责调整储存显示方式、改变底部栏高度，以及打开合成和远程机器入口。

这里的按钮比较小，但都是高频控制。熟悉之后，你可以不用打开设置面板，就能快速调整底部栏的显示状态。


## 排序方式

`S` 按钮用于切换储存空间的排序方式。

常见排序包括数量、模组和名称。当前排序方式会显示在按钮旁边，例如 `Qty`、`Mod` 或 `Name`。

## 升序和降序

`A` / `D` 按钮用于切换升序或降序。

如果你按数量排序，升序会把数量少的物品放前面，降序会把数量多的物品放前面。

如果你在找大量建筑材料，通常用数量降序更快；如果你在找缺口材料，数量升序更容易看到快用完的物品。

## 调整底部栏高度

`+` 和 `-` 按钮用于调整底部栏高度。

需要一次看到更多储存物品时，点击 `+` 增加高度。

需要更多世界视野时，点击 `-` 降低高度。

这个设置只影响 RTSBuilding 的底部栏，不会改变原版背包界面大小。

## 打开合成面板

`C` 按钮用于打开合成面板。

解锁合成终端后，点击 `C`，或直接按键盘 `C`，可以查看当前绑定储存里能够制作的物品。

如果没有看到 `C` 按钮，通常说明合成终端还没有在生存技能树里解锁。

## 远程机器槽

`C` 附近的小 `+` 槽位用于绑定远程机器界面。

解锁远程 GUI 后，点击一个小 `+` 槽位，再左键点击世界里的机器，即可把这个机器界面绑定到底部栏。

绑定后，点击对应槽位可以从 RTS 界面远程打开机器，不需要走回机器旁边。

适合绑定工作台、机器、储罐、处理线输入端等经常需要查看或补料的方块。
