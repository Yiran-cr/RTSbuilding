# 操作指南总览

这套文档面向正在游玩 RTSBuilding 的玩家。你可以从“开始使用”进入，也可以按左侧目录直接查看具体界面区域。

## 推荐阅读顺序

1. 先看“如何下载”，选择与你的 Minecraft 版本和加载器匹配的文件。
2. 再看“开始使用”，完成第一次打开 RTSBuilding 和储存绑定。
3. 接着看“常见操作”，直接学习远程建造、容器存取和仅提取绑定。
4. 想理解界面区域时，再看“顶部栏”“底部栏”和“侧边悬浮窗”。
5. 如果服务器或整合包启用了生存平衡，再看“生存模式机制”。
