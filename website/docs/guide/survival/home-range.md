# RTS 家园与范围限制

生存平衡开启后，RTS 家园决定你可以在哪个范围内使用 RTS 功能。

如果游戏提示距离太远、无法建造、无法远程操作，优先检查 RTS 家园位置和当前技能范围。


## 设置 RTS 家园

打开背包，点击屏幕顶部的“RTS 家园”按钮，进入家园界面。

首次使用时点击“设置初始家园”，回到 RTS 视角后左键点击目标方块确认。右键或 `Esc` 取消。

设置后，RTSBuilding 会以家园位置为中心计算可用范围。家园界面会显示当前家园坐标、维度和建造范围。


## 范围上限与技能进度

RTS 操作范围受技能树进度限制。刚开始时范围较小，随着相关能力解锁，范围会逐步扩大。

完全解锁后的最终范围上限，由设置面板中的“最大操作半径”决定。

## 移动家园

解锁“野外部署”前，请慎重选择初始家园。

解锁“野外部署”后，家园按钮会允许迁移家园。迁移后，新的家园位置会成为 RTS 功能范围的中心。

## 多人队伍共享家园

开启“队伍共享”后，同队伍玩家会共享 RTS 家园锚点。

如果服务器启用了生存平衡和队伍共享，建议队伍先确定基地中心，再设置家园，避免不同成员的操作范围不一致。
