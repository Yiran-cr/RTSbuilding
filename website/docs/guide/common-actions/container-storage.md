# 容器物品与 RTS 存储区的存取

本文说明打开箱子、机器或工作台界面时，如何在容器、玩家物品栏和 RTS 储存之间移动物品。

## 准备

操作前先绑定目标 RTS 储存。

若未绑定，按 `G` 打开 RTSBuilding 界面，点击顶部栏的“储存绑定”按钮并选择目标箱子、机器或储存系统。

打开容器后，界面旁会显示 RTS 侧边悬浮窗。若未显示，按 `G` 打开 RTSBuilding，点击右上角齿轮按钮，在设置面板中开启“容器悬浮窗”。


## 在容器中取出物品放入 RTS 储存

鼠标左键拿起容器里的物品，点击悬浮窗里的“导入”区域。

物品会进入当前绑定的 RTS 储存。


## 把 RTS 中的物品存入容器

在侧边悬浮窗中搜索或翻页，找到 RTS 储存里的物品。

左键点击物品，取出一组到鼠标上。右键点击物品，取出一个。

拿到物品后，按原版容器操作把它放入目标容器或玩家物品栏。


## 把手上的物品送回 RTS 储存

鼠标上拿着物品时，点击 RTS 悬浮窗里的储存物品格，或点击返回/导入区域。

左键会送回一组，右键会送回一个。

## 批量把物品导入 AE 系统

先把 AE 系统绑定为 RTS 储存。

打开旧箱子或机器缓存箱，点击悬浮窗上的“Shift 存入”开关。

对容器或背包里的物品执行 `Shift + 右键`，物品会直接进入当前绑定的 RTS 储存。当前绑定储存是 AE 系统时，这一步就会把物品批量送入 AE。

整理完后，再次点击“Shift 存入”开关，恢复原版 Shift 物品移动操作。


## 移动悬浮窗

点击悬浮窗的“折叠/展开”按钮，隐藏或显示面板。

鼠标左键长按并拖动悬浮窗，自定义它的位置。
