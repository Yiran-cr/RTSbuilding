# Remote Building

Use this flow after you have bound storage and want to place blocks quickly from the RTS view.

## Choose Materials

1. Open RTSBuilding.
2. Open Storage Space in the bottom bar.
3. Search or choose a category to find the block you want.
4. Click the block to make it the current placement item.

The top hint shows the currently selected item.

## Normal Placement

Stay in Interact mode.

Aim at a block face in the world, then right-click to place the selected block.

If you want to place on a chest, machine, or other interactable block, hold `Shift` while right-clicking so you do not accidentally open its screen.

## Quick Build

If you want to place a row, wall, area, or larger shape, open Quick Build in the top bar.

1. Click the Quick Build button.
2. Choose a shape.
3. Choose a fill mode.
4. Follow the top hint and click blocks in the world to set point A, point B, or height.
5. When the preview is correct, right-click to confirm.

If the preview is wrong, click Quick Build again to cancel.

## Undo and Redo

Use `Ctrl + Z` to undo the last group of blocks you built.

Use `Ctrl + Y` to redo it.

Undo and redo usually require the currently selected item to match the block used for that build action.

## Return to Empty Hand

When you are done building and want to open chests, use machines, or interact normally, click the Empty Hand button in the bottom hotbar.

This prevents right-click from placing blocks by mistake.
