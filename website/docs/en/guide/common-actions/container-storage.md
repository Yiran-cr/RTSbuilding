# Containers and RTS Storage

This page explains how to move items between containers, the player inventory, and RTS storage while a chest, machine, or crafting screen is open.

## Preparation

Bind the target RTS storage first.

If nothing is bound, press `G` to open RTSBuilding, click the Storage Binding button in the top bar, and choose the target chest, machine, or storage system.

After opening a container, the RTS side floating window should appear beside the screen. If it does not appear, press `G`, open the gear settings, and enable Container Floating Window.

## Move Items from a Container into RTS Storage

Pick up an item from the container with left click, then click the Import area in the floating window.

The item will be inserted into the currently bound RTS storage.

## Move Items from RTS Storage into a Container

Search or page through the side floating window to find an item in RTS storage.

Left-click an item to take a stack onto the cursor. Right-click it to take one item.

After taking the item, place it into the target container or player inventory using normal container controls.

## Return Cursor Items to RTS Storage

When you are holding items on the cursor, click a storage slot in the RTS floating window, or click the return/import area.

Left click returns a stack. Right click returns one item.

## Bulk Import into an AE System

Bind the AE system as RTS storage first.

Open an old chest or machine buffer, then enable Shift Store on the floating window.

Use `Shift + right click` on items in the container or inventory. They will be sent directly into the current RTS storage. If that storage is an AE system, this bulk-imports the items into AE.

When you are done, click Shift Store again to restore vanilla shift item movement.

## Move the Floating Window

Click the collapse/expand button to hide or show the panel.

Hold and drag the floating window to customize its position.
