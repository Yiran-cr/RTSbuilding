# Side Floating Window

When you open an inventory, chest, crafting table, or machine screen, the side floating window appears beside it.

It has two main parts:

- Storage bay: view the current RTS storage and move items between containers, inventory, and RTS storage.
- Craft button: quickly craft items. It works like the bottom bar crafting panel, but the entry is beside the container screen so you can craft while organizing items.

## When It Appears

Enable or disable Container Floating Window in the RTS settings panel.

## Collapse and Expand

Click Collapse to hide the window. Click Expand to show it again.

## Move the Floating Window

Hold the drag button to move the floating window. Its position is saved automatically.

## Storage Bay: Move Items

The storage bay shows the current bound RTS storage. Treat it like a small RTS storage panel attached to the container screen for quick taking, storing, and organizing.

To take items from RTS storage, click an item icon in the storage bay, then place it into the inventory, chest, or machine slot.

To store items into RTS storage, left-click an item to pick it up, then click the Import area in the floating window.

## Shift Store

Click Shift Store in the floating window to enable the mode.

After enabling it, use `Shift + right click` on an item in an inventory, chest, or machine screen to send it directly into the current RTS storage.

This is useful for organizing chests, clearing material buffers, or moving many items into an AE network after binding it.

Click the button again to disable the mode.

## Search and Sort in the Storage Bay

- Search: type text in the search bar to filter items.
- Sort: click the sort button to order by quantity, mod, or name.
- Direction: click the direction button to switch ascending or descending order.
- Pages: click page buttons to view more items.
- Refresh: click refresh to sync the latest storage data.

## Craft Button

The Craft button in the side floating window opens a quick crafting entry.

It is useful while organizing chests, refilling machines, or remote building. If you notice that you are missing planks, slabs, torches, or a basic material, you can craft from the side floating window without switching back to the bottom bar.

The Craft button uses materials from the current bound RTS storage. Open it, search for the target item, choose the amount, and confirm. RTSBuilding then tries to craft from available materials.

If the list does not show the item you want, check whether the current storage contains the materials, whether the item is craftable in the current game, and whether the search text is correct.
