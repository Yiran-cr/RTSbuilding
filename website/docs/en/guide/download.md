# Download

RTSBuilding is available on Modrinth and CurseForge.

## Official Download Pages

- [Modrinth: RTS building: Build From Above](https://modrinth.com/mod/rts-building-build-from-above/versions)
- [CurseForge: RTS building: Build From Above](https://www.curseforge.com/minecraft/mc-mods/rts-building-build-from-above)

After opening a download page, choose the file that matches your Minecraft version and loader.

## Supported Versions

The main supported lines are:

- Minecraft `1.21.1` / NeoForge
- Minecraft `1.20.1` / Forge

If you are playing on a server or in a modpack, follow that server or modpack's required Minecraft version and loader.

## Search in Launchers

If you use PCL or HMCL, you can also search in the launcher's mod download page:

```text
RTSbuilding
```

If that does not find it, try:

```text
RTS building
```

or:

```text
RTS building: Build From Above
```

Always check that the downloaded file matches your Minecraft version and loader.

## Manual Installation

If you download a `.jar` file from the website, place it in the `mods` folder of the target instance or modpack.

After launching the game, press `G` to open RTSBuilding.
