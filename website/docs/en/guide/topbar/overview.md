# Top Bar Overview

The top bar sits at the top of the RTSBuilding interface. It is the first row of buttons you see after entering RTS mode.

The top bar mainly does three things:

- Switch the current RTS mode.
- Open or close common feature panels.
- Show current state, such as what is selected, what storage is bound, and what you should do next.

## Top Bar Sections

From left to right, the top bar is roughly split into three groups.

The first group contains mode buttons. This is where Interact, Storage Binding, Funnel, Rotate, and similar buttons appear. After clicking one of these buttons, your next clicks in the world use that mode.

The second group contains feature buttons. This includes Quick Build, Ultimine, chunk display, tutorials, and similar tools. They usually open a panel or toggle an overlay rather than changing the whole RTS mode.

The third group is on the right. Click the gear icon to open RTS control settings.

If some buttons are missing, that usually means the feature is locked or disabled by configuration. You can use the buttons that are currently visible.

## What Highlighted Buttons Mean

A highlighted button means that feature or mode is active.

For example:

- Interact is highlighted when you are in normal interaction or placement mode.
- Storage Binding is highlighted when clicks on containers bind them as RTS storage.
- Quick Build is highlighted when the quick build panel is open.
- Chunk display is highlighted when chunk borders are visible in the world.

If you are unsure what state you are in, check the highlighted top bar button and then read the hint text below it.

## Hint Text Under the Top Bar

There are two lines of hint text below the buttons. They change as you interact.

The first line usually tells you:

- The current mode.
- The currently selected placement item.
- Whether a fluid is selected.
- Whether you are empty-handed.

The second line usually tells you:

- Whether storage is bound.
- Whether mined drops auto-store.
- Whether funnel mode is enabled.
- The current build shape.
- The current fill mode.
- The current rotation angle.
- How many build groups can be undone or redone.

If you are in a multi-step action, the hint text also tells you what comes next: setting point A, setting point B, confirming a build, or waiting for you to click a machine for remote GUI binding.

## When to Check the Top Bar

Check the top bar first when:

- You do not know what left click or right click will do.
- You selected an item but are not sure it is active.
- You bound storage but are not sure it worked.
- Quick Build is open and you need to confirm the current shape or fill mode.
- A button disappeared and you want to know whether the feature is still locked.

You do not need to stare at the top bar all the time. Treat it like the dashboard of RTS mode: glance at it before acting to confirm the current state.
