# Settings Panel

Click the gear button in the upper-right corner of the RTS interface to open the settings panel.

## RTS UI Scale

Click the `-` / `+` buttons in the panel to adjust the size of the bottom bar, top bar, and floating window.

This only changes RTSBuilding panels. It does not change the scale of vanilla inventory screens.

## Sensitivity

Click the `-` / `+` buttons to adjust RTS camera movement, rotation, and zoom speed.

After changing it, drag inside the RTS view to test the feel.

## Auto-Store

When Auto-Store is enabled, items mined in RTS mode try to enter bound storage first.

When disabled, mined drops fall into the world.

Turn this off when organizing inventory or when you want to avoid storing items into the wrong target.

## Shift Store

When Shift Store is enabled, hold `Shift` and right-click items in inventory, chest, or machine screens to send them directly into the current RTS storage.

This is useful for bulk transfer. After connecting an AE system, enable Shift Store and quickly move chest materials into AE. It also helps when organizing old base storage, clearing build material boxes, or collecting machine outputs.

When disabled, `Shift + right click` no longer triggers RTS storing.

## Container Floating Window

When Container Floating Window is enabled, opening a chest, crafting table, or machine shows the RTS side floating window beside the screen.

When disabled, the side floating window is hidden.

## Camera Settings

Common options include:

- Start the camera above the player.
- Smooth RTS camera.
- Invert horizontal drag.
- Invert vertical drag.

If drag direction feels wrong, toggle the corresponding invert option.

If camera movement feels harsh, enable Smooth RTS Camera.

## Damage Warnings

In survival, enable hit sound or auto-exit RTS at low health so your player body is not attacked unnoticed while you are in the RTS view.

## Server Rules

Multiplayer survival rules such as build range, material cost, and skill unlocks follow the server or mod configuration.

For detailed survival unlock and home range behavior, read Survival Systems.

Modpack authors and server admins can continue with Modpack and Server Config.
