# Guide Overview

These docs are for players using RTSBuilding in game. Start with First Steps, or use the sidebar to jump directly to a specific UI area.

## Recommended Reading Order

1. Read Download first and choose the file that matches your Minecraft version and mod loader.
2. Read First Steps to open RTSBuilding for the first time and bind storage.
3. Read Common Actions to learn remote building, container transfer, and extract-only binding.
4. Read Top Bar, Bottom Bar, and Side Floating Window when you want to understand each UI area.
5. Read Survival Systems if your server or modpack enables survival balancing.
