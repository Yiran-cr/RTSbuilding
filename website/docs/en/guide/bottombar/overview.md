# Bottom Bar Overview

The bottom bar sits at the bottom of the RTSBuilding interface. It is the main area for choosing materials, browsing storage, using blueprints, and crafting remotely.

You can think of it as several areas: Left Controls, Storage Space, Creative Item Picker, Blueprint Space, and Crafting Panel.

## Bottom Bar Sections

- Left Controls: adjust sorting, bottom bar height, crafting entry, and remote machine bindings.
- Storage Space: view available items, player hotbar, RTS quick slots, recent items, and fluid storage.
- Creative Item Picker: Creative-mode-only item selection from vanilla and modded creative tabs.
- Blueprint Space: import, select, preview, move, rotate, place, and save blueprints.
- Crafting Panel: search craftable items and craft them using materials from bound storage.

Tabs appear above the bottom bar. Click Storage Space, Creative, or Blueprint Space to switch between item storage, creative item picking, and blueprint tools. The Creative tab appears only in Creative mode.

## What to Read First

If you only want to know where each bottom bar button is, read Left Controls first.

If you want to use blocks, tools, quick slots, recent items, or fluids, read Storage Space.

If you are in Creative mode and want to choose materials directly from creative tabs, read Creative Item Picker.

If you want to import building blueprints, pin previews, adjust position, and place them, read Blueprint Space.

If you want to craft directly from bound storage, read Crafting Panel.

## Hover Tooltips

Hover over items, fluids, or buttons to see their tooltips.

Item tooltips appear near the left side of the bottom bar so you can read enchantments, durability, mod item details, and vanilla tooltip information.

If you are unsure what a slot does, hover first, then click.
