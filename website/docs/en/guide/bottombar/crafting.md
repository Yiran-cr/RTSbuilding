# Crafting Panel

The crafting panel lets you view and craft items inside RTSBuilding.

It shows craftable items based on currently available materials. You can craft common blocks or items from bound storage without leaving the RTS view.

## Open the Crafting Panel

Look at the left side of the bottom bar.

After unlocking the crafting terminal, click the `C` button or press the keyboard `C` key to open the crafting panel.

If there is no crafting entry, the feature is usually still locked.

## Search Craftable Items

After opening the panel, type an item name in the search box.

The list shows craftable items that match the search. Click an item to choose how many to craft.

If the desired item is missing, clear the search and check whether the current bound storage has enough materials.

## Choose Crafting Amount

Right-click an item icon in the list to open the amount selection window.

Type the desired amount, or use the buttons in the interface to adjust it.

After confirming, RTSBuilding crafts from available materials. The crafted items go into your inventory or into the bound chest.

## Missing Materials

If materials are not enough, the crafting panel shows which materials are missing.

Return to Storage Space and check whether the materials are in bound storage. Add them, then return to the crafting panel and craft again.

## JEI Integration

If JEI is installed, you can check a target item's recipe in JEI first, then return to RTSBuilding and search for that item in the crafting panel.

For supported recipes, the crafting panel shows whether the item can be crafted and which materials are still missing.

If you cannot find an item, check whether the chest contains the correct materials, whether the item can be crafted in the current game, and whether the search text is spelled correctly.
