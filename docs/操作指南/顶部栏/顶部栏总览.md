# 顶部栏总览

顶部栏位于 RTSBuilding 界面的最上方。进入 RTS 模式后，你会先看到这一排按钮。


顶部栏主要负责三件事：

- 切换当前 RTS 模式。
- 打开或关闭一些常用功能面板。
- 显示当前状态，让你知道现在选中了什么、绑定了什么、下一步该做什么。

## 顶部栏分成哪些区域

从左到右看，顶部栏大致分为三组。

第一组是模式按钮。这里会出现交互、储存绑定、漏斗、旋转等按钮。点击这些按钮后，你接下来对世界的点击方式会改变。

第二组是功能按钮。这里会出现快速建造、连锁挖掘、区块显示、教程等按钮。它们通常不会改变整个 RTS 模式，而是打开一个面板，或切换一个辅助显示。

第三组在右侧，是设置按钮。点击齿轮图标可以打开 RTS 控制设置。

如果某些按钮没有出现，通常表示这个功能还没有解锁，或者当前配置没有启用。你可以先使用已经显示出来的按钮。


## 按钮高亮是什么意思

被高亮的按钮表示它正在生效。

例如：

- 交互按钮高亮时，你处于正常交互/放置模式。
- 储存绑定按钮高亮时，你点击世界中的容器会把它设为 RTS 储存。
- 快速建造按钮高亮时，快速建造面板已经打开。
- 区块显示按钮高亮时，世界中会显示区块边界。

如果你不确定自己现在处于什么状态，先看顶部栏的高亮按钮，再看按钮下方的提示文字。

## 顶部栏下面的提示文字

按钮下方有两行提示文字。它们会随着你的操作变化。

第一行通常告诉你：

- 当前模式是什么。
- 当前选中的放置物品是什么。
- 当前是否选中了流体。
- 当前是否是空手。

第二行通常告诉你：

- 是否已经绑定储存。
- 挖掘掉落物是否会自动入库。
- 漏斗是否开启。
- 当前建造形状是什么。
- 当前填充方式是什么。
- 当前旋转角度是多少。
- 当前可撤销/重做的整组建造数量。

如果你正在进行某个分步骤操作，提示文字也会告诉你下一步。例如设置 A 点、设置 B 点、确认建造，或者正在等待你点击世界中的机器来绑定远程界面。


## 什么时候看顶部栏

下面这些情况，先看顶部栏通常最快：

- 你不知道现在左键或右键会做什么。
- 你刚刚选了物品，但不确定是否真的选中了。
- 你绑定了储存，但不确定绑定成功没有。
- 你打开了快速建造，但不知道当前形状和填充方式。
- 你发现某个按钮不见了，想确认是不是功能还没解锁。

顶部栏不需要一直盯着看。它更像 RTS 模式里的仪表盘：操作前看一眼，确认状态对不对。
