# 模式和按钮

顶部栏里的按钮会改变 RTSBuilding 的操作方式。最重要的是左侧的模式按钮：你点了哪个模式，接下来点击世界里的方块就会触发对应的动作。


## 交互

交互是默认模式，也最接近原版游玩。

在交互模式下：

- 右键可以使用方块、机器、箱子、门，或放置当前选中的物品。
- 左键可以挖掘方块。
- 如果你在底部栏选中了方块，对准世界中的方块右键，就会直接将选中的方块放置上去。
- 如果不能放置，右键会回到正常互动，比如打开机器或箱子。

如果想在箱子或机器等本身可以互动的方块上直接放置物品，请按住 `Shift` 再右键。

你也可以对准世界中的方块按下“选取方块”键，默认是鼠标中键，把它直接选为接下来要放置的物品。


## 储存绑定

储存绑定用于告诉 RTSBuilding：你要从哪个容器或储存系统里拿东西。

使用方法：

1. 点击顶部栏的储存绑定按钮。
2. 对准箱子、桶、抽屉、仓库控制器或其他储存方块。
3. 左键点击它，把它绑定为可存入、可提取的 RTS 储存。
4. 绑定成功后，顶部提示文字会显示已绑定的储存名称。

右键也可以绑定储存，但右键绑定是“仅提取”。这种绑定适合只希望 RTSBuilding 从那里拿东西，不希望它把物品存回去的容器。

进入储存绑定模式后，普通的右键互动会被暂时禁用。这样你对准机器或箱子点击时，就不会误打开操作界面或误放方块。


## 漏斗

漏斗模式用于在 RTS 视角下规划物品的运输路线。它适合需要把物品送到某个目标、或在储存和机器之间搬运物品的时候使用。

使用方法：

1. 点击顶部栏的漏斗按钮。
2. 对准你想操作的目标。
3. 按当前提示进行点击。

如果你只是临时需要漏斗，可以按住快速漏斗热键。默认是 `F`。按住时进入漏斗操作，松开后回到之前的模式。


## 旋转

旋转模式用于从 RTS 俯视镜头下改变方块朝向。

使用方法：

1. 点击顶部栏的旋转按钮。
2. 对准需要旋转的方块。
3. 右键点击方块。

这个模式适合调整楼梯、机器朝向、部分可旋转方块等。能否旋转取决于方块本身是否支持旋转。


## 快速建造

快速建造用于一次放置一组方块。它适合修路、铺地、拉墙、画圆、搭盒子等重复建造。

使用方法：

1. 点击顶部栏的快速建造按钮，打开快速建造面板。
2. 在面板中选择形状。
3. 选择填充方式，例如填满、空心或骨架。
4. 按照顶部提示文字，在世界中设置点位。
5. 预览正确后，右键确认建造。

不同形状需要的步骤不同：

- 单块：直接放置。
- 直线、方形、墙、圆：先设置 A 点，再设置 B 点。
- 立方体：先设置 A 点和 B 点，再调整高度。

如果预览不对，可以再次点击快速建造按钮，或关闭快速建造面板来取消当前预览。

刚建好的整组方块可以用 `Ctrl + Z` 撤销，用 `Ctrl + Y` 重做。撤销和重做通常需要当前选中的物品和建造时一致。


## 连锁挖掘

连锁挖掘用于一次挖掉一组相连的同类方块。

使用方法：

1. 点击顶部栏的连锁挖掘按钮，打开连锁挖掘面板。
2. 设置最多挖掘多少个方块。
3. 对准一个方块并按住挖掘键，默认是左键。
4. 等第一个方块被挖掉后，相连的同类方块会被一次性全部挖掉。

连锁挖掘面板里可以用 `-`、`+`、`MIN`、`MAX` 调整数量，也可以点击数字直接输入。

如果你提前松开了挖掘键，连锁挖掘就不会生效。必须等第一个方块被彻底挖掉，整组方块才会被一起破坏。


## 区块显示

区块显示按钮会显示或隐藏区块边界。

这个功能适合规划农场、机器阵列、基地模块和需要按区块边缘对齐的建筑。

再次点击按钮可以关闭区块显示。

## 教程按钮

顶部栏里的 `i` 按钮会打开当前区域的教程说明。

如果你不知道某个按钮或面板怎么用，可以先点 `i`。教程弹窗会覆盖在 RTS 界面上方，关闭后可以继续操作。

## 设置按钮

右侧齿轮按钮会打开 RTS 控制设置。

这里可以调整 RTS UI 缩放、镜头手感、自动入库、Shift 存入、平滑镜头、伤害提示等选项。

这里的设置只影响你个人的操作习惯和画面效果。建造是否消耗物品、服务器规则等生存平衡限制，不受这些设置影响。
