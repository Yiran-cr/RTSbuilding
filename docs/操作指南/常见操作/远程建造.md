# 远程建造

这个流程适合你已经绑定储存，并想从 RTS 视角快速放置方块。

## 选择材料

1. 打开 RTSBuilding。
2. 在底部栏进入“储存空间”。
3. 搜索或点击分类，找到要用的方块。
4. 点击方块，把它设为当前放置物品。

顶部提示文字会显示当前选中的物品。


## 普通放置

保持交互模式。

对准世界中的方块面，右键放置选中的方块。

如果要在箱子或机器等可以互动的方块上放置，请按住 `Shift` 再右键，这样就不会误打开它们的界面。


## 使用快速建造

如果你要放置一整排、一面墙或一个区域，可以打开顶部栏的快速建造。

1. 点击快速建造按钮。
2. 选择形状。
3. 选择填充方式。
4. 根据顶部文字的提示，点击方块来设置 A 点、B 点或高度。
5. 预览正确后右键确认。

如果预览不对，再次点击快速建造按钮可以取消。


## 撤销和重做

刚建好的整组方块可以用 `Ctrl + Z` 撤销。

如果撤销后又想恢复，可以用 `Ctrl + Y` 重做。

注意：撤销和重做时，你需要保持当前选中的物品和当时建造用的方块是一样的。

## 切回空手

建造结束后，如果你想开箱子、使用机器或普通互动，点击底部栏快捷栏里的空手按钮。

这样右键就不会误放方块，可以正常打开箱子或操作机器了。
